Basic usage

Care 2X is 100% web browser based. You can use it as if you were surfing the web.
The main window is divided into three major blocks as shown:

  • Block A = The main menu. Every work starts here.
  • Block B = The space for working.
  • Block C = Module's or function's name and functional buttons.

Starting a function
Simply click the function's name
Selecting your working language
  • Click the language selection box on the bottom part of the main menu.
  • Click the selected language.
  • Click the "Change" button.
Authenticating yourself
  • Enter your username.
  • Enter your password.
  • Click the "Continue..." button.
